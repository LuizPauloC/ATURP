# 🌎 ATURP - Pancas Guia Turístico

Um website moderno e responsivo para a Associação de Turismo de Pancas (ATURP), contém informações sobre atrativos turísticos, hospedagem e guia completo da região.

## ✨ Características

- 📱 **Design Responsivo** - Otimizado para dispositivos mobile, tablet e desktop
- 🎨 **Interface Moderna** - Design clean e intuitivo
- 🌐 **Multilíngue** - Suporte para múltiplos idiomas
- 📍 **Páginas Dinâmicas** - Páginas criadas dinamicamente com JS

## 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização avançada e responsive design
- **JavaScript** - Interatividade e funcionalidades dinâmicas
- **JSON** - Gerenciamento de dados de páginas
